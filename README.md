# Student Grievance Portal

## Description
This is a React-based frontend application that allows students to submit and track their grievances. Admins can manage grievances by updating their status and deleting them.

---

## Features

### Student
- Register and Login
- Submit grievances
- Track grievance status
- Submit anonymously

### Admin
- View all grievances
- Change status (Pending → In Progress → Resolved)
- Delete grievances

---

## Tech Stack
- React.js
- Tailwind CSS
- React Router

---

## How to Run

1. Clone the repository  
2. Open the project folder  
3. Run:

npm install  
npm run dev  

---

## Limitations
- No backend integration  
- Data resets on refresh
